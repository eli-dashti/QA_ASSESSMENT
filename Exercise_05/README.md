
## How to use

### Installation
For install the framework, you should follow the following steps:

1. ``` 
    git clone ------
2. ```
    npm install
### Run in local

```
    npm test
```
With awesome report:
```
npm run test:awesome
```

### Run with Docker

```
docker build -t elaheh/apiautomation .
```
```
docker run --name api -d elaheh/apiautomation
